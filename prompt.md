«Умный поиск по документам компании» (RAG-система)
Что делает:
Вы загружаете PDF/Word-документы (инструкции, регламенты, техподдержку), и бот отвечает на вопросы по ним с указанием источника.

Почему это ценно для TELEOFIS:
У них наверняка есть внутренняя документация по оборудованию и ПО. Такой проект показывает, что вы умеете внедрять RAG — ключевой навык для промышленного AI.

Технический стек (минимальный):

Язык: Python

LLM: любая open-source (Llama 3, Mistral, Qwen) или GPT через API

Векторная БД: Qdrant / FAISS / Chroma

Эмбеддинги: sentence‑transformers (multilingual) или OpenAI embeddings

Фреймворк: LangChain / LlamaIndex

Интерфейс: web
